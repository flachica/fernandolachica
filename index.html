Entre mis habilidades cuento con

Implantador de Odoo

En mi labor de Intra-emprendedor, fomenté el uso de este sistema de gestión para mejorar el desempeño del departamento de administración para una gran empresa sevillana. Por este motivo fui contratado por una de las mayores implantadoras de Odoo en la que desarrollo mi trabajo a día de hoy.

Backend

Domino lenguajes para su uso en servidor como Python y PHP. Conozco y aplico frameworks como Django o Yii. Un campo que no para de avanzar y al que le aplico numerosas horas de estudio. También domino algo de fronted. Mi framework favorito es VueJS.

SQL

Diseño y normalizo la estructura de los modelos de los sistemas y programo la lógica de negocio en T-SQL/PL-SQL. Optimizo el rendimiento de las consultas con el uso adecuado del lenguaje SQL.
